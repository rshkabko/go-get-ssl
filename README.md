# DOCS

## installation
Add this teposetory to composer.json

```
"repositories": [
        {
            "type": "git",
            "url": "https://github.com/rshkabko/go-get-ssl.git"
        }
    ],
```


Add require to composer.json
```
"rshkabko/go-get-ssl": "dev-master"
```

And update composer
```
composer update
```
